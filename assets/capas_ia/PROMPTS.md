# Capas IA — Prompts para o Gemini (nano banana)

## Como usar (uma única vez)

1. Abra [gemini.google.com](https://gemini.google.com) e cole o prompt do tema.
2. Baixe a imagem gerada.
3. Salve **nesta pasta** (`assets/capas_ia/`) com o nome exato do tema:
   `geral.png`, `futebol.png`, `animais.png`, `brasil.png`, `culinaria.png`, `musica.png`, `natureza.png`
   (aceita também .jpg / .jpeg / .webp)
4. Pronto — toda apostila daquele tema usa a arte automaticamente, para sempre.
   Sem a imagem, a capa sai num design alternativo 100% CSS (também bonito).

⚠️ Regras que já estão embutidas nos prompts (não altere):
- **Sem nenhum texto/letras na imagem** — título, nível e marca entram por cima, nítidos, via código.
- **Terço inferior mais calmo/escuro** — é onde o título é sobreposto.
- Formato retrato (proporção de capa A4).
